Java Toybox
================================================================================

Toybox with assortment of Java practice exercises.


Requirements
--------------------------------------------------------------------------------

- Java 21


Maze
--------------------------------------------------------------------------------

Maze builder with left-hand rule and right-hand rule route explorers.

```bash
./mvnw clean package && java -jar maze/target/maze-1.0-SNAPSHOT.jar 40 20
```
